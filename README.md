# Code Name Coliseum

Coliseum is a SaaS product for Freelance Devlopers and Software Development Agencies to manage clients and projects.

## MVP Features

- Tenant system
- Tenants can create new admin users
- Tenants can create clients
- Tenants can create client users
- Tenants can create projects for clients
- Clients can request feature and bug change requests
- Tenants can turn review change requests
- Tenants can convert change requests into change orders
- Tenants can track time on change orders
- Tenants can connect Coliseum with their stripe account
- Tenants can invoice customers
- Tenants can connect change orders to invoices
- Tenants can charge clients a monthly fee
- Communication happens with simple markdown editor
